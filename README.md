# Autonomous AI Engineering Agent 🚀

Цей репозиторій містить автономного ШІ-агента, здатного самостійно писати, тестувати та виправляти Python-код (Self-Correction Loop). Агент спеціалізується на вирішенні інженерних, математичних та дата-аналітичних задач.

## 🌟 Можливості
- **REPL (Read-Eval-Print Loop):** Агент пише код і самостійно запускає його в ізольованій пісочниці.
- **Самовиправлення:** Якщо код видає помилку компіляції чи логіки, агент аналізує Traceback і переписує код доти, поки він не запрацює.
- **Візуалізація:** Здатність будувати складні графіки (через matplotlib) та автоматично зберігати їх як `.png` файли в директорію проекту.
